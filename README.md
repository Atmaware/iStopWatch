# iStopWatch! 知止观 - Official Website

The official landing page for iStopWatch!, a digital mindfulness stopwatch that transforms screen time into rhythmic breathing cycles.

## Overview

iStopWatch! (知止观) helps you find balance between digital and physical life by breaking your screen time into breathwork-inspired cycles with four phases:

- **住息 (Hold Off)**: Screen awaits - prepare to begin
- **入息 (Inhale)**: Screen is active - use mindfully
- **顿息 (Hold On)**: Screen freezes - wrap up your tasks
- **出息 (Exhale)**: Device locks - move your body

## Website Features

- **Animated breathing cycle visualization** - Live demonstration of the app's core concept
- **Responsive design** - Works beautifully on desktop and mobile
- **Smooth animations** - Fade-in effects and parallax scrolling
- **Philosophy section** - Explains the Confucian and Zen inspirations
- **Feature showcase** - Highlights key app capabilities
- **Platform indicators** - Shows upcoming Android, iOS, and macOS support

## Local Development

Simply open `index.html` in your browser:

```bash
open index.html
```

Or serve it with a simple HTTP server:

```bash
python3 -m http.server 8000
# Visit http://localhost:8000
```

## Deployment

This site is designed to be hosted on GitHub Pages:

1. Push changes to the `main` branch
2. Enable GitHub Pages in repository settings (source: main branch)
3. Site will be available at `https://istopwatch.github.io`

## Files

- `index.html` - Main landing page with semantic HTML structure
- `styles.css` - Modern, minimalist styling with zen-inspired aesthetics
- `script.js` - Interactive breathing cycle animation and scroll effects
- `favicon.svg` - Circular breathing ring icon

## Technologies

- Pure HTML, CSS, and JavaScript (no frameworks)
- CSS Grid and Flexbox for responsive layouts
- CSS animations and transitions
- Intersection Observer API for scroll animations
- Canvas-based SVG for breathing visualization

## Design Philosophy

The website reflects the app's minimalist, mindful aesthetic:
- Clean, spacious layouts
- Zen-inspired color palette (purples, blues, greens)
- Smooth, non-intrusive animations
- Clear hierarchy and typography
- Mobile-first responsive design
